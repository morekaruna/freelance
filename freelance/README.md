Freelance Bidding Platform (Mini Fiverr Clone) - Minimal MERN Starter

Features:
- User signup/login (client/freelancer)
- Clients can post jobs
- Freelancers can view jobs and place bids
- JWT auth, MongoDB storage

Run locally:
1. Backend:
   cd backend
   cp .env.example .env
   npm install
   npm run dev

2. Frontend:
   cd frontend
   npm install
   npm run dev
